WP Simple Backup is a plugin to take backup of website codebase and database.

== Description ==

WP Simple Backup is a plugin to take backup of website codebase and database and store it to defined location on server.

== Installation ==

1. Upload `wpsimplebackup.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= A question that someone might have =

What wpsimplebackup plugin does ?

Using wpsimplebackup plugin you can backup code, database on one click.
